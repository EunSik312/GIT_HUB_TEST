# 👁️ AI 및 컴퓨터 비전 실습 강의 (충남인력개발원, 2024)

이 저장소는 2024년 충남인력개발원에서 진행된 **AI 및 컴퓨터 비전 기반 실습 강의** 자료를 정리한 것입니다.

총 20회차에 걸쳐 Python 기초부터 OpenCV, 딥러닝, YOLO, Segmentation, 자율주행 등 다양한 실습을 포함합니다.
